## UOL联合开放实验室项目
### 使用说明
####  一、登陆
- 登陆网址：http://http://localhost:8080/J1012Web/main.jsp
- 用户名：`xiaocao`密码：`xiaocao`
- 用户名：`lala`密码：`123`

#### 二、SQL文件
1. 包含内容：
	- 表结构及其数据
	- 视图
	- 序列
2. 执行方式
	- 第一步：打开数据库客户端链接上数据库
	- 第二步：右键点击数据库表，选择导入数据，选择提供的sql文件进行导入

#### 三、服务器部署项目
1. 找到war包
2. 找到电脑安装的tomcat目录
3. 将war包直接放入webapps下
4. 找到bin目录下的startup.bat批处理文件，双击启动tomcat
5. 打开浏览器访问项目
6. 找到bin目录下的shutdown.bat批处理文件，双击关闭tomcat
